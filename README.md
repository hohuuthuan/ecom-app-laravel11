## Run Locally

Clone the project

```bash
  git clone https://github.com/hohuuthuan/ecom-app-laravel11.git
```

Go to the project directory

```bash
  cd ecom-app-laravel11
```

Install dependencies

```bash
  composer install & npm install 
```

Start the app

```bash
  npm run app
```


## Tech Stack

**Server side rendering:** Laravel 11, Html, Css, JavaScript, Boostrap 5, Composer, NPM
