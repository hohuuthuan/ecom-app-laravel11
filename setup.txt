### HƯỚNG DẪN CÀI ĐẶT DỰ ÁN LẦN ĐẦU ###
- Tạo file .env và copy nội dung .env.example
- Chỉnh sửa tên DB_DATABASE, DB_USERNAME và DB_PASSWORD
- Vào thư mục econ-app-laravel11 mở lên terminal
- Cài đặt dependencies: composer install && bun install
- Link thư mục: php artisan storage:link
- Chạy migrate database sạch và seed 2 role cơ bản: php artisan migrate:fresh --seed
- Chạy role seeder: php artisan db:seed --class=RoleSeeder
- Chạy dự án: bun run app

### HƯỚNG DẪN LẤY CODE MỚI ###
- Vào thư mục econ-app-laravel11 mở lên terminal
- Chạy lệnh: git pull origin main
- Chạy lệnh (nếu có chỉnh sửa database, không có thì bỏ qua): php artisan migrate:fresh --seed
- Chạy dự án: bun run app
